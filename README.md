# Game-of-Words
This is a simple game in which the player gets 5 unique letters and he/she has to make the longest possible english using only these 5 letters.

The player can repeat the letters as many times as it he/she wants for forming his/her answer word.
While the letters are given to the player, he/she can choose whether he/she wants a vowel or a consonant.

The scoring system is based on the length of the word entered by the player, if it is a valid word, the player gets points equal to the length of the word, else the player gets 0 points.

The game is made by deploying a server-client architechture using Socket Programming in Python.
The GUI is made using Pygame library of Python.
